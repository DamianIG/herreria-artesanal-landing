# Herrería Artesanal Formosa — Landing Page

Landing page informativa para **Herrería Artesanal Formosa**, diseñada para presentar servicios, trabajos realizados y canales de contacto.

## Objetivo
- Presentar el negocio de forma clara y profesional.
- Facilitar el contacto inmediato por WhatsApp y llamadas.
- Mostrar trabajos realizados mediante una galería visual.
- Incluir opiniones de clientes con carrusel.

## Secciones incluidas
- Top Bar (mensaje + accesos rápidos)
- Header + navegación (desktop y menú mobile)
- Hero (presentación y llamados a la acción)
- Nosotros
- Servicios
- Galería (solo imágenes)
- Opiniones (carrusel Bootstrap)
- Contacto + Mapa (ubicación pendiente de confirmar)
- Botón flotante de WhatsApp

## Tecnologías
- HTML5
- TailwindCSS (CDN)
- Bootstrap 5 (solo carrusel)
- JavaScript (menú mobile + año del footer)

## Configuración
No requiere instalación.

1. Abrir `index.html` en el navegador  
   o
2. Usar Live Server (VS Code) para previsualización local.

## Pendiente
- Confirmar ubicación exacta para Google Maps (dirección/coordenadas del cliente) y reemplazar el `iframe`.
